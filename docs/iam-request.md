# 【RIZAP ご担当者様へ】セットアップにあたってのお願い

コンディションチェックAIエージェント（日次コメント自動生成）を、RIZAP様側のGoogle Cloud
プロジェクトで動かすにあたり、下記のご対応をお願いいたします。

## ① 対象GCPプロジェクトの確認

- 既存の `rizap-marketing` プロジェクト（BigQuery・既存分析エージェントが稼働中）を
  利用します（新規プロジェクトの払い出しは不要になりました）
- 正式なプロジェクトIDをご共有ください

## ② セットアップ担当へのアクセス権付与

READMEのコマンドを実行してセットアップを行う担当者に、
上記プロジェクトの **編集者（Editor）** 権限を付与してください。

- 付与先: `＿＿＿＿＿＿＿＿＿＿＿＿@＿＿＿＿＿` （実行担当のGoogleアカウントを別途ご連絡します）

## ③ Vertex AIでのGemini利用有効化

Gemini（Google純正モデル）をVertex AI経由で呼び出します。Vertex AI API
（`aiplatform.googleapis.com`）の有効化は `deploy/setup_gcp.sh` で自動的に行うため、
基本的に追加のご対応は不要です。組織側でAPI有効化に制限がかかっている場合のみ、
解除のご協力をお願いする可能性があります。

## ④ Drive/Sheetsアクセス用のGoogleアカウント

被験者コンディションチェックのスプレッドシートを読み書きするため、対象データに
アクセスできるGoogleアカウントでの認可（1回のみ、ブラウザでログイン・許可）が必要です。
このアカウントは既にREHATCH側でもログイン情報を把握しているとのことなので、
RIZAP様側で追加対応いただく必要は基本的にありません（念のため共有します）。

---

上記①②が整い次第、`deploy/setup_gcp.sh` → `deploy/deploy.sh` の実行でセットアップが完了します
（詳細は `docs/runbook.md`）。
